# Bus Display - Image Build Guide

Create flashable image like Pwnagotchi.

## Quick Start (Manual)
1. Flash Pi OS Lite
2. Clone repo to /home/pi/bus-display
3. Run: sudo ./install.sh
4. Reboot

## First Boot
- Shows boot screen on e-ink
- Detects network
- Shows setup instructions with IP
- Web UI auto-starts at :5000

## Image Build (Automated)
Use pi-gen or custom build script (see docs).
